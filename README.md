# python_projects

# 🧑‍💻 Python Mini Projects Collection

This repository contains multiple beginner-friendly Python projects demonstrating core programming concepts such as OOP, file handling, and database connectivity.

---

## 📌 Projects Included

### 1. 🏦 Bank Account System
A simple banking system built using Python OOP concepts.

**Features:**
- Create account with name and account number
- Deposit money
- Withdraw money with balance validation
- Check current balance

**Concepts Used:**
- Classes & Objects
- Methods
- Conditional statements

📂 File: `Bank account system.py`

---

### 2. 💼 Employee Salary Calculator
A Python program to calculate employee salary including allowances.

**Features:**
- Input employee details
- Calculate Dearness Allowance (DA)
- Add House Rent Allowance (HRA)
- Compute total salary

**Concepts Used:**
- Object-Oriented Programming (OOP)
- Functions & calculations

📂 File: `Employee salary calculator.py`

---


### 3. 🧾 Student Record File System (CSV Handling)
A file-based student record system using CSV.

**Features:**
- Create CSV file
- Append new student records
- Read and display data

**Concepts Used:**
- File Handling
- CSV module
- Exception handling

📂 File: `student record file system.py`

---

### 4. 🎓 Student Management System (Basic Python)
A simple system to manage student details and grades.

**Features:**
- Input student details
- Display information
- Calculate grades based on marks

**Concepts Used:**
- Classes & Objects
- Conditional logic

📂 File: `student management system.py`
